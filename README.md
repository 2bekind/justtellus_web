# Telegram Bot - Приватные личные сообщения

Бот для пересылки сообщений в формате приватных личных сообщений.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Заполните `.env` файл:
   - `BOT_TOKEN` - токен вашего бота (получите у @BotFather в Telegram)
   - `OWNER_ID` - ваш Telegram ID (можно узнать у @userinfobot)

## Запуск

```bash
python bot.py
```

## Как это работает

Когда кто-то пишет боту, сообщение пересылается вам в формате:
```
[ {username} ]

[ {text} ]
```

## Получение токена бота

1. Найдите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в файл `.env`

## Получение вашего Telegram ID

1. Найдите @userinfobot в Telegram
2. Отправьте любое сообщение
3. Бот вернет ваш ID
4. Скопируйте ID в файл `.env` в поле `OWNER_ID`
